# Product Overview

This is a personal portfolio and blog website for Omar Maaouane Veiga, hosted at [omar-fyi.com](https://omar-fyi.com).

## Purpose

- Showcase personal projects and professional work
- Publish technical blog posts and articles
- Provide an online presence with minimal cost and maximum performance

## Key Features

- Static site generation for fast page loads
- Blog with markdown-based content management
- Content collections with schema validation
- Custom 404 page
- SEO-optimized with sitemap generation
- Syntax highlighting for code blocks

## Design Philosophy

- Minimal and fast
- Cost-effective (domain + ~$1-2/month AWS)
- Fully automated deployment
- Infrastructure as code
- Security-first (private S3, HTTPS everywhere, TLS 1.2+)
